# Contratos de prueba

Tres pares de JPGs estilo "documento escaneado" que simulan documentos legales reales en español. Cada par consiste en un **contrato original** y su **enmienda / adenda**, tomados del corpus de ejemplo que provee el proyecto final del Módulo 4 del bootcamp.

Los cambios esperados que se listan abajo salen del documento de ground-truth del bootcamp (`Diferencias entre Contratos y Enmiendas de ejemplo.docx`) y se reproducen acá para que la demo se pueda cruzar sin salir del repo.
